# Проект : Билеты на событие

**Описание проекта**

В проекте можно: 
 - выбрать направление поездки; 
 - выбрать время поездки; 
 - выбрать тип и количество билетов на поездку.

**Стек**

Стек: React, CSS, БЭМ, Git.
